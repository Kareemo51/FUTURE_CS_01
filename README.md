# FUTURE_CS_01 - Vulnerability Assessment Report

## Overview

This project presents a vulnerability assessment conducted on a live web application using passive security testing techniques.

## Target

http://zero.webappsecurity.com

## Tools Used

* Nmap
* OWASP ZAP (Passive)
* Browser Developer Tools

## Key Findings

* Insecure HTTP connection
* Missing security headers
* Information disclosure
* Vulnerable JavaScript libraries
* Information leakage

## Methodology

The assessment was conducted using non-intrusive, read-only methods to identify common security weaknesses without exploiting the system.

## Repository Contents

* Report: Contains the full vulnerability assessment report
* Screenshots: Evidence supporting the findings
